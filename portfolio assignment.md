### Academy Cinemas
This is a responsive web application built using **MVC**, **HTML5**, **CSS3**, and **Bootstrap**. It mimics a modern movie theater interface, showcasing current showtimes, movie details, and an interactive layout. It highlights my core front-end integration skills alongside C# back-end routing.
* **Tech Stack:** C#, ASP.NET MVC, Bootstrap, JavaScript, HTML5, CSS3

### Entity Framework Student Console App
A backend data application utilizing **Entity Framework 6** and a **Code-First workflow**. The application programmatically defines a database schema using C# domain models, automatically generates a local SQL Server database (`SchoolDB`), inserts a user record, and saves changes.
* **Tech Stack:** C#, Entity Framework 6, LINQ, SQL Server (LocalDB)
